# 鼠标悬停图片剪辑路径效果

#### 介绍

一款css鼠标悬停图片剪辑路径效果，当鼠标悬停在图片上时，图片按一定路径放大的动画特效，生动形象，让图片灵活起来。

#### 技术栈

HTML + CSS + JS + DOM

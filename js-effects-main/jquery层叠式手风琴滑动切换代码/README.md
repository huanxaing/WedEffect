# jquery层叠式手风琴滑动切换代码

#### 介绍

jquery层叠式手风琴滑动切换代码，基于[WOW.js](https://www.delac.io/WOW/docs.html)默认设置元素动画，鼠标点击可以滑动切换图片。注：[WOW.js是Animate.css的黄金搭档](https://blog.csdn.net/sunyctf/article/details/128534064)

#### 技术栈

HTML + CSS + JS + JQuery + WOW.js
